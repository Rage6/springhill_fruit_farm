# Springhill Fruit Farm

### http://springhill-fruit-farm.com

This website is for the Springhill Fruit Farm, which is a family-owned orchard in Shiloh, Ohio. It was my second freelance website design/development, and was made entirely out of front-end code. I used the preprocessor SASS when writing the CSS in order to adapt the layout for desktops, laptops, and mobile devices. In order to ease Javascript development, I used jQuery throughtout the script. The final result is an effective, easy-to-use interface that allows clients to find the information that they want quickly.

Aside from its index page, it's pages are:

## Fruit & More (http://springhill-fruit-farm.com/produce.html)
Here is where all of their different types of apples, peaches, pears, etc. are listed. It initially lists ALL of the farm's produce, but my Javascript allows a user to narrow it down to one type by a menu beneath the main title. Each box shows the name of that product, its characteristics, and when it will be ripe

## Where & When (http://springhill-fruit-farm.com/directions.html)
This is where a user finds the orchard's location, hours of operation, and opening/closing dates. Worth mentioning is the iframe element with a Google map that allows users to interact with that map. A link there also quickly takes the user to the Google link where the orchard's address is automatically inserted as the "final destination" on Google Maps.
